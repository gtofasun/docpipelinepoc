Figures and Graphics

Figures are used to express information graphically. They can be inserted with or without a caption. Typically graphics used to illustrate a concept include a caption, while screen captures used in a procedure do not.

Storing your Graphic Images

Store all of the images associated with a project in the Content > Resources > Images folder. For very large projects, you can create sub-folders in the Images folder, if necessary, to organize the images.

Storing all of your graphics in Content/Resources/Images is recommended but not required. You can store your graphics anywhere in the Contents folder. For example, you might want to store all of the graphics for a book in the folder used to store the rest of the content for the book.

Graphic Formats

There are currently no technical documentation graphic standards at SafeNet. It is strongly recommended, however, that any images you use in your project can be updated. That is, always store the editable source file in the Images folder along with the image that is used in the documentation.

Sizing Your Graphics

To ensure that your graphics fit within the 7 inch page width of the PDF page, scale all of your graphics so that they are no wider than 7in x 96pixel/in = 672 pixels. 650 pixels is a good round number to standardize on.

GIMP is a good free open-source graphics editing tool that you can use to resize bitmap images. For best results with screen captures, resize your bitmaps using Sinc (Lanczos3) interpolation (Image > Scale Image > Quality).

Inserting Figures

You can insert a figure with or without a caption. Unlike table captions, which are specified in the Insert Table dialog, figure captions are added as a paragraph (p.figureCaption) above the figure.

To insert a figure

Use the p.figureCaption style to enter a caption for the figure, if desired. Use of captions is optional. You must use a caption if you want to insert a cross-reference to the figure.

Select Insert > Image.

The Insert Image dialog is displayed.

Select the image and choose the resize options as required.

Enter a screen tip, if desired.

HTML Top-Navigation Outputs

This section describes the HTML top-navigation outputs and how to use them in your projects. It contains the following topics:

Overvew

Configuring the Top-Navigation Outputs

Overvew

HTML Top-Navigation Outputs use the HTML_FRAMELESS skin. This skin displays the TOC for the target at the top of the window. Because in does not use frames, which can trigger security warnings in vulnerability scans, this skin is well suited to on-product documentation. The HTML_FRAMELESS skin provides a more modern look, although navigation can be difficult if your TOC contains more than two levels, or multiple topics. By default, the HTML_FRAMELESS skin includes the Breadcrumbs proxy. For improved navigation, you can use different master pagesto add the Topic Toolbar and/or Menu proxies to your outputs, as described and illustrated below.

If you use the Topic Toolbar or Menu proxies, you must use the HOME master page for your home page(s) that include the p.productLine and p.title styles. See Configuring the Top-Navigation Outputs for details.

Proxies

Breadcrumbs Adds breadcrumbs links above the content.
Topic Toolbar Adds buttons for print, remove highlighting, previous topic, and next topic to the top and bottom of each page. Other buttons are also available. You can edit the proxy to customize which buttons are included.
Menu Adds a context-sensitive topic menu to the left of the content.

Building the Example Frameless HTML Outputs Examples

You can build Framelesss HTML output by building the targets in the Project/Targets/HTML_FRAMELESSS folder, as follows:

XHTML_FRAMELESS_SINGLE_DOC Builds the output described in The HTML_FRAMELESS skin with the FRAMELESS master page
XHTML_FRAMELESS_MULTI_DOCS_TOOLBAR Builds the output described in The HTML_FRAMELESS skin with the FRAMELESS_TOOLBAR master page
XHTML_FRAMELESS_MULTI_DOCS_MENU Builds the output described in The HTML_FRAMELESS skin with the FRAMELESS_MENU master page

The targets will also tell you which TOC is used to build the output. See Configuring Your Output Targets and TOCs: Defining the Output Contents, Structure, and Layout for more information.

Configuring the Top-Navigation Outputs

The top-navigation outputs use the proxies described below. To configure your outputs to use the proxies, you must specify the master page and skin to use in your targets and, for the Menu proxy, specify the TOC to use to generate the context-sensitive menu. You also need to set the master page to HOME for your home page topic. See the procedures below for step-by-step instructions.

The Breadcrumbs Proxy

The Breadcrumbs proxy is included in all the top-navigation master pages. It does not require any configuration.

The Topic Toolbar Proxy

The Topic Toolbar proxy is included in the FRAMELESS_TOOLBAR and FRAMELESS_MENU master pages. It is formatted using the FRAMELESS_TOOLBAR skin component (Project/Skins/FRAMELESS_TOOLBAR.flskn). This skin component controls the buttons that are displayed, and the toolbar styles and text. There is also a provision for adding custom Javascript. Please contact the template administrator before editing the skin component, or to provide suggestions for enhancements.

The proxy properties define the skin component used by the proxy. To view the proxy properties, open the FRAMELESS_TOOLBAR or FRAMELESS_MENU master pages, right-click on the topic toolbar proxy and select Properties. You should not need to edit the proxy properties.

The Menu Proxy

The Menu proxy is included in the FRAMELESS_MENU master page only. It is formatted using the FRAMELESS_MENU skin component (Project/Skins/FRAMELESS_MENU.flskn). This skin component controls the look of the menu, and is part of the template design. Do not edit this skin component.

The proxy properties define the TOC used to generate the context-sensitive menu, and some other properties set to their suggested default values, as shown below. To view the proxy properties, open the FRAMELESS_MENU master page, right-click on the topic toolbar proxy and select Properties.

You need to edit the proxy properties to specify the TOC you want to use. This is the TOC for your HTML output. See HTML TOCs for more information.

Configuring your outputs to use the HTML_FRAMELESS skin with the FRAMELESS master page

This configuration includes the Breadcrumbs proxy only. You need to configure your target.

To configure your outputs to use the HTML_FRAMELESS skin with the FRAMELESS master page

Configure your target as described in Creating the HTML Target, ensuring that the following options are set:

Skin tab General > HTML_FRAMELESS
Advanced tab Master Page > FRAMELESS

Configuring your outputs to use the HTML_FRAMELESS skin with the FRAMELESS_TOOLBAR master page

This configuration includes the Breadcrumbs and Topic Toolbar proxies. You need to configure your target and home page.

To configure your outputs to use the HTML_FRAMELESS skin with the FRAMELESS_TOOLBAR master page

Configure your target as described in Creating the HTML Target, ensuring that the following options are set:

Skin tab General > HTML_FRAMELESS
Advanced tab Master Page > FRAMELESS_TOOLBAR

Edit the properties for your home page (that uses the p.productLine and p.title styles) to use the HOME master page:

Navigate to your home page topic file.

Right-click on the file and select Properties.

Set Topic Properties > Masterpage: to HOME

Configuring your outputs to use the HTML_FRAMELESS skin with the FRAMELESS_MENU master page

This configuration includes the Breadcrumbs, Topic Toolbar, and Menu proxies. You need to configure your target, home page, and Menu proxy properties.

To configure your outputs to use the HTML_FRAMELESS skin with the FRAMELESS_MENU master page

Configure your target as described in Creating the HTML Target, ensuring that the following options are set:

Skin tab General > HTML_FRAMELESS
Advanced tab Master Page > FRAMELESS_MENU

Edit the Menu proxy properties to specify the TOC to use for the contest-sensitive menu:

Navigate to Contents/Resources/MasterPages.

Open the FRAMELESS_MENU master page, right-click on the Menu Proxy and select Properties. The following dialog is displayed.

Specify the TOC to use. This is the TOC for your HTML output. See HTML TOCs for more information.

Edit the other Menu Content properties if required. It is recommended that you use the default values shown above. Do not change the skin file.

Edit the properties for your home page (that uses the p.productLine and p.title styles) to use the HOME master page:

Navigate to your home page topic file.

Right-click on the file and select Properties.

Set Topic Properties > Masterpage: to HOME

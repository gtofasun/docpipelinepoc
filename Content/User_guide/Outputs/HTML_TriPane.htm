HTML Tri-Pane Outputs

The HTML tri-pane outputs use the HTML_FRAMES skin. This skin uses frames to displays the TOC for the target in a frame to the left of the content, and a frame above for the logo and search bar. Because it provides superior navigation, it is recommended that you use the frames skin for large projects. See Configuring Your Output Targets for information about applying skins to your targets.

Building the Example Frames HTML Output

You can build Frames HTML output by building the Project/Targets/HTML_FRAMES/XHTML_FRAMES_MULTI_DOCS target. The target will also tell you which TOC is used to build the output. See Configuring Your Output Targets and TOCs: Defining the Output Contents, Structure, and Layout for more information.

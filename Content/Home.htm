The Gemalto Customer Documentation Template documentation, linked below, is provided in HTML and PDF formats. See the README for more information. You can download the latest version from the Gemalto Customer Support Portal at https://supportportal.gemalto.com.

Templates
Flare Template User Guide PDF Describes the template elements and the workflow used to create single-sourced documentation in HTML and/or PDF formats.
Note Template PDF Used for release notes and other technical notes.
Technical Note Template PDF Deprecated. Differs from the Note template only by the inclusion of the subjectNotes subtitle paragraph on the title page. Since the subject of the technical note can usually be conveyed by the title alone, rather than using both the title and subject, it is recommended that you use the simpler Note template for all technical notes.
Upgrading
Upgrade Guide PDF Provides an overview of the new features in the template and step-buy-step instructions for upgrading to the new template.
